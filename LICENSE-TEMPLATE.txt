===
TERMS

This software is licensed under GPLX, which combines:
- GPL freedoms (use, modify, distribute)
- MIT option (closed source path available)
- Fairness (sustainable maintainer support)

Default: GPL-style (share modifications)
Optional: MIT-style (pay Closed Path rate to keep private)

===
FAIRNESS FRAMEWORK

This project uses XBasic-1.0 fairness framework.
See FAIRNESS.txt for contribution tiers and terms.

Framework summary:
- Free for companies under $500k revenue
- Tiered contribution above threshold
- Caps to prevent hostage situations
- Exempt: non-profits, public education, government, startups, contributors

===
GOOD FAITH

GPLX is built on mutual good faith from all parties.
Corporations are not inherently bad, maintainers are not inherently victims.
We are partners in building software that benefits humanity.

All parties agree to:
- Honest communication
- Reasonable interpretation of terms
- No gaming the system
- No hostage situations (from either side)
- Assume good intent first

If standard terms create genuine hardship, parties may negotiate
alternative arrangements.

===
ENFORCEMENT

1. Communicate before any enforcement action
2. Negotiate in good faith

4. Pursue legal action only as absolute last resort
